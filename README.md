# SSG Lifting Club 🏋️‍♂️

Welcome to the **SSG Lifting Club** (formerly known as Bar Is Loaded)! This is a premium, lightning-fast web application built specifically for powerlifters and strength athletes to calculate barbell loads, track RPE, and manage plate inventories.

## Features ✨

* **Load Bar (Calculator):** Easily calculate exactly which plates you need to load on the bar for a target weight.
* **Reverse Calculator:** Tap the plates you see on the bar to instantly calculate the total weight.
* **Custom Inventory Management:** Set exactly how many plates of each weight you own. The calculator is smart enough to only suggest plates you actually have in your inventory!
* **RPE & 1RM Calculators:** Built-in tools for autoregulation. Input your last set to get an estimated 1-Rep Max (e1RM), and calculate the target weight for your next set.
* **Meet Prep:** Quick access to resources for overcoming plateaus in the Squat, Bench, and Deadlift.
* **Premium UI:** A sleek, responsive dark mode design with glassmorphism effects, dynamic animations, and easy unit toggling (KG/LB).

## Tech Stack 🛠️

This app is built for maximum speed and simplicity, requiring zero build steps or complex frameworks:
* **HTML5**
* **Vanilla JavaScript**
* **Vanilla CSS (Custom Properties, Flexbox, Grid)**
